API Documentation

GET /api/resource

Description

Retrieve an array of resources.

Request URL

https://example.com/api/resource

Response

[
    {
        "id": 1,
        "name": "Resource Name",
        "description": "Description"
    }
]

POST /api/resource

Description

Create a new resource.

Request URL

https://example.com/api/resource

Body Parameters

{
    "name": "New Resource",
    "description": "Description of the new resource"
}
