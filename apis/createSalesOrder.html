POST /company/{company_id}/v1/api/sales_orders

Description

Create a new sales order within a company. Provide sales order details including customer information, order amounts, and payment details.

Request URL

https://getflex.solutions/company/{company_id}/v1/api/sales_orders

Request Body Example

{
    "customer_id": 10,
    "order_date": "2021-01-01T00:00:00",
    "ship_date": "2021-01-05T00:00:00",
    "status": "pending",
    "total_amount": 500.00,
    "currency": "USD",
    "payment_method": "credit card",
    "notes": "Urgent delivery"
}

Sample Response

Response: {"message": "Sales order created successfully", "order_id": 2}
