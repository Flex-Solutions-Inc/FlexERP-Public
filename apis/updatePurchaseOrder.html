PUT /company/{company_id}/v1/api/purchase_orders/{purchase_order_id}

Description

Update details of an existing purchase order. This can include changing the expected delivery date, order status, or the total amount.

Request URL

https://getflex.solutions/company/{company_id}/v1/api/purchase_orders/{purchase_order_id}

Request Body Example

{
    "expected_delivery_date": "2023-11-01T00:00:00Z",
    "status": "completed",
    "total_amount": 2200.00
}

Sample Response

{
    "message": "Purchase order updated successfully",
    "purchase_order_id": 123
}
