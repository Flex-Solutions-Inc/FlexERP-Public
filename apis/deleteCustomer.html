DELETE /company/{company_id}/v1/api/customers/{customer_id}

Description

Delete a specific customer from the company by specifying the customer ID. This action cannot be undone.

Request URL

https://getflex.solutions/company/{company_id}/v1/api/customers/{customer_id}

Example Request

https://getflex.solutions/company/123/v1/api/customers/1

Sample Response

Response: {"message": "Customer deleted successfully"}
