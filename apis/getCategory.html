GET /company/{company_id}/v1/api/categories/{category_id}

Description

Retrieve details of a specific category by providing the company ID and category ID.

Request URL

https://getflex.solutions/company/{company_id}/v1/api/categories/{category_id}

Example Request

https://getflex.solutions/company/123/v1/api/categories/1

Sample Response

{
    "category_id": 1,
    "company_id": "123",
    "name": "Electronics",
    "description": "Devices and gadgets",
    "parent_category_id": null,
    "is_main": true,
    "thumbnail_image": "path/to/image.jpg",
    "active": true,
    "created_at": "2021-01-01T00:00:00",
    "updated_at": "2021-01-02T00:00:00"
}
