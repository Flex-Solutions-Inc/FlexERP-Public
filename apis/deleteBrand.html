DELETE /company/{company_id}/v1/api/brands/{brand_id}

Description

Delete a specific brand from the company by specifying the brand ID. This action cannot be undone.

Request URL

https://getflex.solutions/company/{company_id}/v1/api/brands/{brand_id}

Example Request

https://getflex.solutions/company/123/v1/api/brands/2

Sample Response

Response: {"message": "Brand deleted successfully"}
