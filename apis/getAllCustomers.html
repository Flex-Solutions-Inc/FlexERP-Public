GET /company/{company_id}/v1/api/customers

Description

Retrieve all customers associated with a specified company. This endpoint lists all customers under the given company ID.

Request URL

https://getflex.solutions/company/{company_id}/v1/api/customers

Example Request

https://getflex.solutions/company/123/v1/api/customers
