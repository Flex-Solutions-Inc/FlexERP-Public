GET /company/{company_id}/v1/api/products

Description

Retrieve a list of all products for a specified company. The company ID must be provided in the URL path to fetch the related products.

Request URL

https://getflex.solutions/company/{company_id}/v1/api/products

Example Request

https://getflex.solutions/company/12345/v1/api/products
