GET /company/{company_id}/v1/api/brands/{brand_id}

Description

Retrieve details of a specific brand by providing the company ID and brand ID.

Request URL

https://getflex.solutions/company/{company_id}/v1/api/brands/{brand_id}

Example Request

https://getflex.solutions/company/123/v1/api/brands/1

Sample Response

{
    "brand_id": 1,
    "company_id": "123",
    "name": "Brand One",
    "description": "Description of Brand One",
    "created_at": "2021-01-01T00:00:00",
    "updated_at": "2021-01-02T00:00:00"
}
