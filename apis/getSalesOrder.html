GET /company/{company_id}/v1/api/sales_orders/{order_id}

Description

Retrieve details of a specific sales order by providing the company ID and order ID.

Request URL

https://getflex.solutions/company/{company_id}/v1/api/sales_orders/{order_id}

Example Request

https://getflex.solutions/company/123/v1/api/sales_orders/1
