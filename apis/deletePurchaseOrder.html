DELETE /company/{company_id}/v1/api/purchase_orders/{purchase_order_id}

Description

Delete a specific purchase order by its ID. This action is irreversible and should be used with caution.

Request URL

https://getflex.solutions/company/{company_id}/v1/api/purchase_orders/{purchase_order_id}

Sample Response

{
    "message": "Purchase order deleted successfully",
    "purchase_order_id": 123
}
