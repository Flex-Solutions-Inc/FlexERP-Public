POST /v1/api/parts

Description

Create a new part in the inventory. This endpoint is used to add parts to the database, specifying details such as name, description, stock levels, and price per unit.

Request URL

https://getflex.solutions/v1/api/parts

Request Body Example

{
    "part_name": "Hydraulic Pump",
    "description": "A component used in hydraulic systems",
    "on_hand": 150,
    "min_stock_level": 50,
    "price_per_unit": 250.00
}

Sample Response

{
    "message": "Part created successfully",
    "part_id": 123,
    "part_name": "Hydraulic Pump",
    "on_hand": 150,
    "min_stock_level": 50,
    "price_per_unit": 250.00
}
