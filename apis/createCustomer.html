POST /company/{company_id}/v1/api/customers

Description

Create a new customer within a company. Provide customer details including name, contact information, and other relevant data.

Request URL

https://getflex.solutions/company/{company_id}/v1/api/customers

Request Body Example

{
    "first_name": "Jane",
    "last_name": "Smith",
    "email": "jane.smith@example.com",
    "phone": "987-654-3210",
    "address": "456 Maple Avenue",
    "city": "Springfield",
    "state": "IL",
    "postal_code": "62704",
    "country": "USA"
}

Sample Response

Response: {"message": "Customer created successfully", "customer_id": 2}
