POST /company/{company_id}/v1/api/categories

Description

Create a new category within a company. Provide category details including name, description, and whether it's a main category.

Request URL

https://getflex.solutions/company/{company_id}/v1/api/categories

Request Body Example

{
    "name": "Smartphones",
    "description": "Collection of smartphones",
    "parent_category_id": 1,
    "is_main": false,
    "thumbnail_image": "path/to/smartphone.jpg",
    "active": true
}

Sample Response

Response: {"message": "Category created successfully", "category_id": 3}
