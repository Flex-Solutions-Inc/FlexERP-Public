PUT /company/{company_id}/v1/api/customers/{customer_id}

Description

Update existing customer information by providing the company ID and customer ID along with new data for the customer.

Request URL

https://getflex.solutions/company/{company_id}/v1/api/customers/{customer_id}

Request Body Example

{
    "first_name": "John",
    "last_name": "Doe Updated",
    "email": "updated.john.doe@example.com",
    "phone": "123-456-7891",
    "address": "123 Elm Street Updated",
    "city": "Springfield",
    "state": "IL",
    "postal_code": "62701",
    "country": "USA",
    "status": "active"
}

Sample Response

Response: {"message": "Customer updated successfully"}
