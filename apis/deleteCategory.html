DELETE /company/{company_id}/v1/api/categories/{category_id}

Description

Delete a specific category from the company by specifying the category ID. This action cannot be undone.

Request URL

https://getflex.solutions/company/{company_id}/v1/api/categories/{category_id}

Example Request

https://getflex.solutions/company/123/v1/api/categories/2

Sample Response

Response: {"message": "Category deleted successfully"}
