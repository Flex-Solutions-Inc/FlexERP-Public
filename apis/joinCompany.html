POST /v1/api/companies/{company_id}/{uuid}/join

Description

Join an existing company using a company ID and user UUID. Optionally, a company join code can be required.

Request URL

https://getflex.solutions/v1/api/companies/101/UUID123/join

Request Body Example

{
    "company_code": "ABC123"
}

Sample Response

{
    "message": "User successfully joined the company",
    "company_id": 101,
    "uuid": "UUID123"
}
