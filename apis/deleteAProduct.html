DELETE /company/{company_id}/v1/api/products/{product_id}

Description

Delete an existing product by specifying the company ID and product ID in the URL path. This action cannot be undone.

Request URL

https://getflex.solutions/company/{company_id}/v1/api/products/{product_id}

Example Request

https://getflex.solutions/company/12345/v1/api/products/001

Sample Response

Response: Product with ID 001 has been deleted.
