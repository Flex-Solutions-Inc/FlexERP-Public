PUT /company/{company_id}/v1/api/suppliers/{supplier_id}

Description

Update existing supplier information by providing the company ID and supplier ID along with new data for the supplier.

Request URL

https://getflex.solutions/company/{company_id}/v1/api/suppliers/{supplier_id}

Request Body Example

{
    "name": "Updated Supplier One",
    "contact_name": "Johnathan Doe",
    "address": "123 Updated Street, City",
    "phone": "123-456-7890",
    "email": "updated@supplierone.com",
    "website": "http://www.updatedsupplierone.com",
    "status": "active"
}

Sample Response

Response: {"message": "Supplier updated successfully"}
