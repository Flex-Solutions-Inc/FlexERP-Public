DELETE /company/{company_id}/v1/api/suppliers/{supplier_id}

Description

Delete a specific supplier from the company by specifying the supplier ID. This action cannot be undone.

Request URL

https://getflex.solutions/company/{company_id}/v1/api/suppliers/{supplier_id}

Example Request

https://getflex.solutions/company/123/v1/api/suppliers/2

Sample Response

Response: {"message": "Supplier deleted successfully"}
