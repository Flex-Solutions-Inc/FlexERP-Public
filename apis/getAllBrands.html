GET /company/{company_id}/v1/api/brands

Description

Retrieve all brands associated with a specified company. This endpoint lists all brands under the given company ID.

Request URL

https://getflex.solutions/company/{company_id}/v1/api/brands

Example Request

https://getflex.solutions/company/123/v1/api/brands
