POST /v1/api/companies

Description

Create a new company. Provide details including the owner's user ID, company name, size, plan, and a detailed description of the company.

Request URL

https://getflex.solutions/v1/api/companies

Request Body Example

{
    "company_owner_id": 123,
    "company_name": "Example Inc.",
    "company_size": 100,
    "company_plan": 1,
    "company_description": "A leading provider of innovative tech solutions."
}

Sample Response

{
    "message": "Company created successfully",
    "company_id": 101,
    "uuid": 123,
    "company_name": "Example Inc.",
    "company_plan": 1,
    "company_size": 100,
    "company_description": "A leading provider of innovative tech solutions."
}
