PUT /company/{company_id}/v1/api/brands/{brand_id}

Description

Update existing brand information by providing the company ID and brand ID along with new data for the brand.

Request URL

https://getflex.solutions/company/{company_id}/v1/api/brands/{brand_id}

Request Body Example

{
    "name": "Updated Brand One",
    "description": "Updated description of Brand One"
}

Sample Response

Response: {"message": "Brand updated successfully"}
