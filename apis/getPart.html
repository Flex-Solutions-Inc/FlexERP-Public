GET /v1/api/parts/{part_id}

Description

Retrieve detailed information about a specific part by its ID. Useful for inventory checks and updates.

Request URL

https://getflex.solutions/v1/api/parts/123

Sample Response

{
    "message": "Part retrieved successfully",
    "part_id": 123,
    "part_name": "Hydraulic Pump",
    "on_hand": 150,
    "min_stock_level": 50,
    "price_per_unit": 250.00
}
