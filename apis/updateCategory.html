PUT /company/{company_id}/v1/api/categories/{category_id}

Description

Update existing category information by providing the company ID and category ID along with new data for the category.

Request URL

https://getflex.solutions/company/{company_id}/v1/api/categories/{category_id}

Request Body Example

{
    "name": "Updated Electronics",
    "description": "Updated description of electronic devices",
    "parent_category_id": null,
    "is_main": true,
    "thumbnail_image": "path/to/updated_image.jpg",
    "active": true
}

Sample Response

Response: {"message": "Category updated successfully"}
