PUT /v1/api/parts/{part_id}

Description

Update existing part details such as stock levels, price, and description. Essential for maintaining accurate inventory records.

Request URL

https://getflex.solutions/v1/api/parts/123

Request Body Example

{
    "part_name": "Hydraulic Pump",
    "description": "Updated description for better clarity.",
    "on_hand": 200,
    "min_stock_level": 75,
    "price_per_unit": 300.00
}

Sample Response

{
    "message": "Part updated successfully",
    "part_id": 123
}
