GET /company/{company_id}/v1/api/suppliers

Description

Retrieve all suppliers associated with a specified company. This endpoint lists all suppliers under the given company ID.

Request URL

https://getflex.solutions/company/{company_id}/v1/api/suppliers

Example Request

https://getflex.solutions/company/123/v1/api/suppliers
