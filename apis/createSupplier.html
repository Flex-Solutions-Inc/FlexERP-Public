POST /company/{company_id}/v1/api/suppliers

Description

Create a new supplier within a company. Provide supplier details including name, contact information, and other relevant data.

Request URL

https://getflex.solutions/company/{company_id}/v1/api/suppliers

Request Body Example

{
    "name": "Supplier Three",
    "contact_name": "Alice Smith",
    "address": "345 Boulevard, City",
    "phone": "321-654-0987",
    "email": "contact@supplierthree.com",
    "website": "http://www.supplierthree.com",
    "status": "active"
}

Sample Response

Response: {"message": "Supplier created successfully", "supplier_id": 3}
