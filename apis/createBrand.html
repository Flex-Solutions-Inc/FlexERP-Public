POST /company/{company_id}/v1/api/brands

Description

Create a new brand within a company. Provide brand details including name and description.

Request URL

https://getflex.solutions/company/{company_id}/v1/api/brands

Request Body Example

{
    "name": "Brand New",
    "description": "A description of the new brand"
}

Sample Response

Response: {"message": "Brand created successfully", "brand_id": 3}
