GET /company/{company_id}/v1/api/categories

Description

Retrieve all categories associated with a specified company. This endpoint lists all categories under the given company ID.

Request URL

https://getflex.solutions/company/{company_id}/v1/api/categories

Example Request

https://getflex.solutions/company/123/v1/api/categories
