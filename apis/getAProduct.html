GET /company/{company_id}/v1/api/products/{product_id}

Description

Retrieve details of a specific product by providing the company ID and product ID in the URL path.

Request URL

https://getflex.solutions/company/{company_id}/v1/api/products/{product_id}

Example Request

https://getflex.solutions/company/12345/v1/api/products/001
