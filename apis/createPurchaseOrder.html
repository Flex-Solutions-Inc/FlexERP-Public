POST /company/{company_id}/v1/api/purchase_orders

Description

Create a new purchase order for a specific company. This endpoint allows for specifying supplier details, order and expected delivery dates, and the total amount of the order.

Request URL

https://getflex.solutions/company/{company_id}/v1/api/purchase_orders

Request Body Example

{
    "supplier_id": 100,
    "order_date": "2023-10-01T00:00:00Z",
    "expected_delivery_date": "2023-10-15T00:00:00Z",
    "total_amount": 2000.50
}

Sample Response

{
    "message": "Purchase order created successfully",
    "purchase_order_id": 123
}
