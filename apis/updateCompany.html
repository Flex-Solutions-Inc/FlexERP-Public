PUT /v1/api/companies/{company_id}

Description

Update existing company details such as name, size, and plan.

Request URL

https://getflex.solutions/v1/api/companies/101

Request Body Example

{
    "company_name": "Example Inc. Updated",
    "company_size": 150,
    "company_plan": 2
}

Sample Response

{
    "message": "Company updated successfully",
    "company_id": 101
}
