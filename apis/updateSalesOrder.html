PUT /company/{company_id}/v1/api/sales_orders/{order_id}

Description

Update existing sales order information by providing the company ID and order ID along with new data for the sales order.

Request URL

https://getflex.solutions/company/{company_id}/v1/api/sales_orders/{order_id}

Request Body Example

{
    "customer_id": 10,
    "order_date": "2021-01-01T00:00:00",
    "ship_date": "2021-01-10T00:00:00",
    "status": "shipped",
    "total_amount": 550.00,
    "currency": "USD",
    "payment_method": "paypal",
    "notes": "Delivery updated"
}

Sample Response

Response: {"message": "Sales order updated successfully"}
